Kubernetes の基礎を学びます。このモジュールでは、クラスターアクセス、Pod の理解、Pod との対話、スケーリングのためのデプロイメント、Service によるアプリケーションの公開について扱います。実践的なラボと実世界の課題を通じて、これらの必須の Linux スキルを習得します。
